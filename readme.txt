ИНСТРУКЦИЯ ПО ЗАПУСКУ ПРОЕКТА

1. Установка зависимостей:
   pip install -r requirements.txt

2. Выполнение миграций:
   python manage.py migrate

3. Создание суперпользователя (опционально):
   python manage.py createsuperuser

4. Создание тестового меню (опционально):
   python manage.py create_test_menu

5. Запуск сервера:
   python manage.py runserver

После запуска сервера:
- Главная страница: http://127.0.0.1:8000/
- Админка: http://127.0.0.1:8000/admin/

ИСПОЛЬЗОВАНИЕ МЕНЮ В ШАБЛОНАХ:

{% load menu_tags %}
{% draw_menu 'main_menu' %}

Где 'main_menu' - название меню, созданного в админке.

СОЗДАНИЕ МЕНЮ:

1. Войдите в админку Django
2. Перейдите в раздел "Меню"
3. Создайте новое меню с уникальным названием
4. Добавьте пункты меню через раздел "Пункты меню"

Параметры пункта меню:
- Название пункта - отображаемый текст
- Родительский пункт - для создания иерархии
- URL - явный URL (например: /page/)
- Named URL - имя URL из urls.py (например: page1)
- Порядок сортировки - для управления порядком

Приоритет URL: если указан named_url, он имеет приоритет над url.
